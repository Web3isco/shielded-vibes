## How ZK Is Used

Shielded Vibes uses **Groth16 zero-knowledge proofs** to achieve privacy on a public blockchain:

| Operation | What ZK hides | How it works |
|---|---|---|
| **Deposit** | Amount deposited | Create a commitment (UTXO) вҖ” the proof shows "I deposited a valid amount" without revealing it |
| **Transfer** | Sender, recipient, amount | Prove ownership of input notes + produce output notes under recipient's public key вҖ” all hidden |
| **Withdraw** | Which notes were spent | Prove you own unspent notes without revealing which ones or their values |
| **Vibe Notes** | Message content | AES-GCM encrypted with PBKDF2 key derived from recipient's address вҖ” only recipient can decrypt |

**Verification:** The Groth16 verifier is deployed as a Soroban smart contract on Stellar. Every proof is verified on-chain before the Pool contract executes the transaction. No trusted setup leakage, no plaintext amounts visible.

## Setup & Run

**Prerequisites:** Rust, Stellar CLI, Node.js, Freighter browser extension (testnet).

**One command to serve:**
```bash
make serve
```

Then open `http://localhost:8000` in your browser.

**Full setup (first time only):**
```bash
# 1. Deploy contracts to testnet
./deployments/scripts/deploy.sh testnet \
  --deployer <identity> \
  --asp-levels 10 --pool-levels 10 \
  --max-deposit 1000000000 \
  --vk-file deployments/testnet/circuit_keys/policy_tx_2_2_vk.json

# 2. Build circuits
make circuits-build

# 3. Build & serve
make serve
```

> в„№пёҸ After deployment, update contract addresses in `deployments/testnet/deployments.json`.

### Transaction Flow

1. **Deposit** вҶ’ Create commitment (UTXO) вҶ’ Prover Worker generates Groth16 proof вҶ’ submitted to Pool contract
2. **Transfer** вҶ’ Select input notes вҶ’ specify recipient keys вҶ’ compute outputs with ZK proof вҶ’ submitted
3. **Withdraw** вҶ’ Prove ownership of notes вҶ’ generate ZK proof вҶ’ receive tokens to specified address

All amounts, recipients, and note values are hidden by Groth16 proofs verified on-chain by the Soroban verifier contract.
